About

Keep Silent developed by team SWEET TALKERS

Now a days, when a ny employee or student will go for their work whether it is a lecture class or any office meeting. Maximum time they will forgot to put their devices in silent mode. It will create disturbance and careless mistakes of that person. I have also faced this problem in my day to day life. Using machine learning various tools we can identify the human behavior

To overcome this problem we will make an app in which device automatically turn on and off silent mode when required. 1. Using machine learning various tools in the app we can identify the human behaviour and the user shoiuld not worry about to put his device on silent mode 2. User can add the location on the map where he will want to put his device on the silent mode and whenever the user comes under 100m of that area, solent mode will turn on and when he will not in range silent mode goes off
